@classe: 3ESGI AL1

## Description
Mini interpréteur avec AST contenant:
- evalExpr et evalInst
- IF, WHILE ET FOR
- Fonctions :
    - void sans paramètre
    - void avec paramètre
    - paramètres et return
- Gestion du scope des variables
- Récursivité terminale
- Variables globales
